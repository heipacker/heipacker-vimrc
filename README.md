# heipacker-vimrc-

it is easy to install.

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

git clone https://github.com/heipacker/heipacker-vimrc.git

ln -s ~/heipacer-vimrc/.vimrc .vimrc

open .vimrc file

:PluginInstall
